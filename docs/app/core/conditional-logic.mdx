# Conditional Logic

Visibility / enablement updates after `manager.onScreenDataChange` and dynamic passes.

Example:
```ts
manager.onScreenDataChange({ age: 17 });
manager.onScreenDataChange({ age: 18 });
```

Function predicate (if supported by your schema):
```ts
{
  id: 'licenseNumber',
  type: 'text',
  label: 'Driver License #',
  visibleIf: (values: any) => Number(values.age) >= 16
}
```
