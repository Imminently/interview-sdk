# Theming

Import default theme CSS:
```tsx
import '@imminently/interview-sdk-theme-default/dist/theme.css';
```

Dark mode toggle:
```tsx
<button onClick={() => document.documentElement.classList.toggle('dark')}>Toggle</button>
```

Utility overrides:
```tsx
<div className='interview-panel bg-background border rounded p-4 shadow-sm'>
  {/* TODO implement example */}
</div>
```
